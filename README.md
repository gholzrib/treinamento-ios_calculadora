# treinamento-ios_calculadora
Exercicio do treinamento de iOS oferecido pela CWI Software

Criar uma calculadora com
   - display para o resultado;
   - ação de limpar;
   - algarismos de 0 a 9;
   - operações:
       - adição;
       - subtração;
       - divisão;
       - multiplicação;
       - x ao quadrado;
       - raiz quadrada;
